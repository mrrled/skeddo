# 📚 Skeddo

**Система составления расписания для школ дополнительного образования.**

## О проекте

**Skeddo** — десктопное приложение для создания и управления учебными расписаниями. Проект разрабатывался для школы дополнительного образования и ориентирован на работу в условиях нестабильного или отсутствующего интернет-соединения, например во время выездных учебных смен.

Основная цель системы — упростить процесс составления расписания и автоматически выявлять конфликты между преподавателями, аудиториями и учебными группами.

## ✨ Основные возможности

* Создание и редактирование расписаний
* Проверка конфликтов в реальном времени
* Управление преподавателями, аудиториями, предметами и учебными группами
* Поддержка групп и подгрупп
* Экспорт расписания в PDF и Excel
* Автономная работа через локальную базу данных
* Поиск и фильтрация данных
* Работа с несколькими расписаниями одновременно

## 🔍 Система проверки конфликтов

Во время редактирования расписания приложение автоматически проверяет:

* пересечения учебных групп и подгрупп;
* занятость преподавателей;
* занятость аудиторий.

Конфликты отображаются непосредственно в интерфейсе, что позволяет обнаруживать ошибки ещё на этапе составления расписания.

## 🏗️ Что было реализовано

В рамках проекта была разработана полноценная система управления расписанием: от доменной модели и бизнес-логики до пользовательского интерфейса и экспорта документов.

Особое внимание уделялось архитектуре приложения, автономной работе без постоянного доступа к интернету и удобству использования для сотрудников образовательных организаций.
